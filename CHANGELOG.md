# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-17

### Added
- Initial release of Aegis Architectural Governance Engine.
- Agentic Microkernel architecture for polyglot governance.
- Structured rule definition via `.aegis/rules.yaml`.
- Gated enforcement command `aegis check` for CI/CD and pre-commit hooks.
- AI Skills for context-aware governance discovery and evolution.
- Brownfield support via `.aegis/baseline.json` technical debt ledger.
- Automated remediation planning via `aegis apply`.
- Consensus audit trail via `aegis evolve`.
- MCP Server implementation for native agentic integration.
- Full self-governance enforcement for framework development.
